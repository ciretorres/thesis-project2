## Instrucciones:

Descarga la carpeta <a href="Builds.zip" download>Builds.zip</a>, descomprime y ejecuta el archivo de la aplicación según tu sistema operativo `windows-amd64/Prototype_v1.exe`, `linux-amd64/Prototype_v1`, 

ó

1. Descarga e instala [Processing](https://processing.org/download) de la página oficial.

<img src="./captures/Captura de Pantalla 2021-12-27 a la(s) 19.30.21.png" width="800">

2. Clona el repositorio o descarga el ZIP del proyecto de https://github.com/ciretorres/thesis-project en tu computadora :octocat:

  <img src="./captures/Captura de Pantalla 2021-03-24 a la(s) 16.01.03.png" width="800">

3. Descomprime y/o abre el archivo thesis-project/Prototype_v1/Prototype_v1.pde
  
  <img src="./captures/Captura de Pantalla 2021-03-24 a la(s) 14.49.27.png" width="800">

4. Para reproducirlo necesitas instalar e importar la biblioteca de [Peasycam](https://www.mrfeinberg.com/peasycam/#about) desde el IDE de Processing. Selecciona **Sketch > Importar biblioteca...> Añadir biblioteca**:
  
  <img src="./captures/Captura de Pantalla 2021-12-27 a la(s) 19.36.13.png" width="800">

5. Busca e instala la librería [Peasycam](https://www.mrfeinberg.com/peasycam/#about) desde el Contribution Manager como se muestra en la imagen.

  <img src="./captures/Captura de Pantalla 2021-03-24 a la(s) 14.47.04.png" width="800">

6. Una vez instalada e importada la librería. Ejecuta el proyecto.
  
  <img src="./captures/Captura de Pantalla 2021-03-24 a la(s) 14.47.16.png" width="800">

7. Disfruta la experiencia :sparkles:
  
  <img src="./captures/Captura de Pantalla 2021-12-27 a la(s) 19.41.36.png" width="800">
